RAG Simulation (From Scratch)

This project demonstrates a simplified version of Retrieval-Augmented Generation (RAG) without using any external APIs or large language models.

What This Project Shows

Document chunking

Text vectorization (TF-IDF)

Cosine similarity

Retrieval of relevant information

Simulated answer generation

How It Works

Load a document

Split into chunks

Convert chunks into numerical vectors

Convert user query into a vector

Compute cosine similarity

Retrieve most relevant chunk

Generate answer based on retrieved chunk

Installation
pip install scikit-learn
Run
python rag_simulation.py
